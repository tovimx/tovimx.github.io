Deployment Model

The add-on is intended for private distribution within a Google Workspace environment and can be installed for the whole domain, selected organizational units, or selected groups.

Admin Notes

  • Verify the Marketplace listing is configured as private.
  • Review requested OAuth scopes before broad rollout.
  • Install gradually if staged testing is preferred.

Emoji Supercharge Setup

Basic setup guidance for installing and using the Image Replacer Slides add-on.

Hub Admin Config Help

User Setup

  1. Install the add-on through your Google Workspace environment.
  2. Open a Google Slides deck.
  3. Go to Extensions > Image Replacer > Open Image Replacer.
  4. Authorize the add-on if prompted.

Required Access

The add-on uses limited Slides and Drive scopes needed to read the active presentation context and fetch replacement images.

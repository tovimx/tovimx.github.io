Using Image Replacer

Quick instructions for opening the add-on and replacing selected images in Google Slides.

Hub Support Report Issue

Basic Flow

  1. Select one or more images in a Slides deck.
  2. Open Extensions > Image Replacer > Open Image Replacer.
  3. Paste a Drive URL or file ID, or upload a local image.
  4. Run the replace action and confirm the result.

Tip

If nothing happens, re-check that your current selection contains image elements, not text boxes or shapes.
